# tncli releases

Binary releases for [tncli](https://toantran292.github.io/tncli-docs/).

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/toantran292/tncli-releases/main/install.sh | bash
```

## Manual download

Grab the binary for your platform from [Releases](https://github.com/toantran292/tncli-releases/releases/latest).
